# Клеточные автоматы. Лесные пожары

## Состояния клеток:

EMPTY = 0 TREE = 1 BURNING = 2 BURNT = 3

## Основные правила:
1. Горящее дерево на следующем шаге становится сгоревшим
2. Сгоревшее дерево на следующем шаге становится пустой клеткой
3. На пустой клетке может вырасти новое дерево
4. Дерево может загореться от удара молнии
5. Дерево может загореться, если рядом есть горящие соседи

## Дополнительные правила:
1. Вероятность возгорания дерева, если горящий сосед находится с ветреной стороны, увеличивается
2. Огонь активнее передается на деревья находящиеся выше
3. Общая влажность в мире влияет на вероятность возгорания
